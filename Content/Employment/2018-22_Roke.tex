  \item Led DevOps on a COTS software-defined radio programme, extending CI pipelines and adding pre-commit gating so automated build and test became the default rather than an afterthought.
  \item Designed FPGA firmware modules as Design Lead on a \href{https://roke.co.uk/innovations/protecting-personnel-from-ieds}{miniaturised Electronic Counter-Measures radio}, owning module architecture, hardware interfaces and driver software, and building OpenCPI firmware workers in VHDL with VHDL and Python verification. % chktex 12
